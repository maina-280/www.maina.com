2. Discuss the diffrence between client-side and server-side programming

client-side programming
Runs on the user's browser and control the webpage's looks and feel e.g. HTML.CSS, JavarScript
Server -side programming
Runs on a web server and processes data, authentication and database management e.g. Python, Node, PHP,js

3. what is a content management system(CMS) ?.

A content management system is a software that allows users to creat an manage website content without extensive coding knowledge .

Three popular content management system platforms:

  1. Drupal: A highly customizable CMS for complex websites .
  2. Wordpress: The most popular CMS , ideal for blogs and businesses.
  3. Joomla : A flexible CMS for ecommerce and social networks.

CSS ( cascading style sheets ) is used to style HTML elements and enhance a website's appearance .

5. What is Responsive Web Design? .

Responsive Web design ensure a website adapts to different screen sizes for optimal user experience.

Three Responsive Design strategies

  1. Flexible grid layouts
  2. CSS media queries
  3. Flexible image and media
